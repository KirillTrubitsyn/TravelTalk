# TravelTalk

Мультиязычный голосовой переводчик для путешествий. PWA-приложение с поддержкой 5 языков.

## Языки

Русский, бразильский португальский, английский, немецкий, французский.

## Возможности

Четыре режима: текстовый ввод, распознавание текста с фото (OCR), голосовой перевод, режим диалога с автоматической озвучкой перевода.

## Установка на телефон

Открыть сайт в браузере → «Добавить на главный экран».

## API

Для работы необходим ключ Anthropic API. Вводится один раз, хранится локально.
